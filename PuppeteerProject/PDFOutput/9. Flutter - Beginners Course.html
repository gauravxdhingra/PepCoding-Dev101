                    The course content -- "First Application" -- was fine. However, the instructions for setting up the development environment need to be updated. Specifically, the Android SDK's "sdkmanager" will not work for any versions of Java > 8, which has been recently deprecated in favor of Java 11. There is a patch to make it work for Java 9. In order to function with newer versions of Java, the JAVA_HOME environment variable must be set within the "sdkmanager" to point to the older Java 8. Without fixing "sdkmanager", "flutter doctor" will complain about the licenses. After fixing "sdkmanager", "flutter doctor" can be run to get concurrence with the various license provisions. In addition, the "flutter" plug-in must be installed in both Android Studio and IntelliJ IDEA, otherwise "flutter doctor" will complain about missing "flutter" and "dart" plug-ins. All is now working.
